# 🎯 AI Interview Prep Tool

> Practice smarter. Improve faster. Get hired.

An AI-powered interview preparation platform that helps users generate personalized interview questions, simulate real interview experiences, and receive detailed AI-driven feedback on their responses.

Built to make interview preparation more interactive, practical, and data-driven.

---

## 🚀 Live Demo

🌐 Frontend: https://ai-based-interview-prep-tool.vercel.app/login.html

⚙️ Backend API: https://ai-based-interview-prep-tool.onrender.com

---

## ✨ Features

### 🔐 User Authentication
- Secure registration and login
- JWT-based authentication
- Protected routes and user-specific data

### 🤖 AI Question Generation
- Generate interview questions based on:
  - Topic
  - Difficulty level
  - Number of questions
- Powered by OpenRouter AI models

### 🎤 Interview Simulation
- Answer questions in a structured interview environment
- Mimics real interview flow

### 📊 AI Evaluation & Feedback
- Overall interview score
- Performance assessment
- Strengths and weaknesses
- Improvement recommendations

### 📈 Analytics Dashboard
- Interview history
- Performance tracking
- Skill-based insights
- Progress monitoring

### ☁️ Cloud Deployment
- Frontend hosted on Vercel
- Backend hosted on Render
- Database hosted on MongoDB Atlas

## 🔄 How It Works

1. User creates an account
2. User selects interview topic and difficulty
3. AI generates interview questions
4. User answers the questions
5. AI evaluates responses
6. Results are stored in MongoDB
7. Dashboard visualizes performance and progress
